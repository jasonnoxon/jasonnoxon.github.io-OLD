HTML Recipe Layout Project

Now that we have a whole new vocabulary of HTML tags available to us, it's a good idea to practice writing these elements by hand in context. This will also familiarize you with your coding environment and how to preview your code in the browser as your work. The project goal is to create an HTML document of a cooking recipe of your choice. Due to the normal anatomy of a recipe, you'll have to opportunity to try using many different types of HTML elements. Your project must contain:

  • 3 Different Heading Tags
  • Both Types of HTML Lists
  • Bolded Text
  • Paragraphs
  • An Image (optional)

Project 1: Band Site

You made it! We've learned a lot so far about HTML and CSS. While there is a lot left to learn about these two languages, we have to tools needed to build an entire web page from top to bottom.

This project will begin with you creating a homepage. There will be 5 main areas on your homepage:

  • Header
  • Promo Area
  • Tour Date Listing
  • Highlighted Merch Section
  • Footer

Your band name and branding is up to you. While much of this course is technical and code-based. It's important to think about the aesthetics of what we are building. This is a visual job and beginning to identify what colors, fonts, layout styles and overall themes go well with certain types of content will benefit your projects as they progress.
